## 简介<br/> 
基于 Base Admin一套客户信息管理系统<br/> 
按照客户实际需求进行了简单的二次开发
## 技术栈<br/> 
前端：layui<br/> 
java后端：SpringBoot + Thymeleaf + WebSocket + Spring Security + SpringData-Jpa + MySql + mongoDB<br/>
